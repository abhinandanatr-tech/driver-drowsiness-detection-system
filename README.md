# 🚗 Driver Drowsiness Detection System

## 📌 Overview

This project is an embedded system designed to detect driver fatigue in real-time and prevent accidents by alerting the user when signs of drowsiness are detected.

## ⚙️ Features

* Eye blink detection using IR sensor
* Head tilt detection using tilt sensor
* Buzzer alert system
* Low-cost and easy to implement
* Works without internet

## 🧰 Components Used

* IR Sensor
* Tilt Sensor
* Buzzer
* Microcontroller (Arduino / ESP32)
* Breadboard & Jumper wires

## 🔌 Working Principle

The system continuously monitors eye movement and head position:

* If eyes remain closed for a certain duration OR
* If abnormal head tilt is detected

👉 The system triggers an alert (buzzer)
